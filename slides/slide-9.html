list_alt Encabezados Comunes

Encabezado Tipo Descripción
Authorization Solicitud Credenciales para autenticar al cliente.
Content-Type Ambos Indica el tipo de medio del recurso (ej. application/json).
Accept Solicitud Tipos de contenido aceptables para la respuesta.
User-Agent Solicitud Información sobre el software que actúa como cliente.

traffic Códigos de Estado HTTP

2xx check_circle

Éxito

La solicitud fue recibida, entendida y aceptada exitosamente. (ej. 200 OK, 201 Created)

3xx call_split

Redirección

Se necesita tomar más acciones para completar la solicitud. (ej. 301 Moved Permanently)

4xx error

Error del Cliente

La solicitud contiene sintaxis incorrecta o no puede cumplirse. (ej. 400 Bad Request, 404 Not Found)

5xx dns

Error del Servidor

El servidor falló al cumplir una solicitud aparentemente válida. (ej. 500 Internal Server Error)
